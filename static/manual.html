초기 설정 최초 1회

포털을 처음 열면 관리자 계정 생성 → ClassIn API 연결 순서로 진행합니다.

1

관리자 계정 생성

최초 접속 시 자동으로 초기 설정 화면이 나타납니다.
아이디, 비밀번호, 표시이름, 시간대를 입력하고 🔧 초기 설정 버튼을 클릭하세요.

2

API 설정

사이드바 → API 설정 메뉴에서 ClassIn에서 발급받은 SID와 SECRET을 입력합니다.
연결 테스트 버튼으로 정상 연결을 확인하세요. ✅ API 인증 성공! 메시지가 나오면 완료입니다.

3

강사 계정 생성

사이드바 → 계정 관리에서 강사가 포털에 로그인할 계정을 만듭니다.
ClassIn UID는 강사 등록 후 자동으로 나오는 값을 입력하세요.

강사/학생 등록

ClassIn에 사용자를 등록하면 ClassIn 앱에서 로그인할 수 있습니다.

⚠️
전화번호 형식이 중요합니다! 00국가코드-번호(앞0제외)
예) 한국: 0082-1047666192 / 중국: 0086-13912345678
1

강사 관리 → 강사 등록

전화번호, 닉네임, 비밀번호를 입력 → 강사 등록 클릭
성공 시 UID:1234567 형태로 고유번호가 반환됩니다. 이 UID는 이후 코스/수업에 자동 입력됩니다.

2

학생 관리 → 학생 등록

동일 방식으로 학생도 등록합니다. TOPIK 급수, 학습 목적, 모국어를 프로필에 추가 입력할 수 있습니다.

💡
UID 클릭 시 자동 복사됩니다. 다른 폼에서 붙여넣기 할 수 있어요.

코스 → 수업 생성

핵심 운영 흐름: 코스 생성 → 학생 배정 → 수업 생성

1

코스 관리 → 코스 생성

코스 이름과 담당 강사 UID를 입력 → 생성
반환된 코스 ID가 수업 생성, 학생 배정 폼에 자동으로 입력됩니다.

2

학생 배정

코스 ID + 학생 UID → 배정 클릭
배정된 학생은 해당 코스의 모든 수업에 자동 참여합니다.

3

수업/입장 → 수업 생성

코스 ID, 수업명, 시작/종료 시간, 강사 UID → 수업 생성
생성 후 캘린더에 자동으로 일정이 표시됩니다.

4

입장 링크

UID를 입력하고 🔗 링크 생성 → 강사/학생이 ClassIn 앱 없이 웹으로 입장 가능한 URL이 생성됩니다.

교육과정 등록

레벨별 단원, 핵심 교육 포인트를 체계적으로 등록합니다.

1

레벨 선택

초급 / 중급 / 고급 / 비즈니스 중 선택

2

단원 정보 입력

단원 번호, 제목, 설명을 입력합니다.
예) 1과 — 인사하기

3

핵심 교육 포인트

줄바꿈으로 구분하여 여러 포인트를 입력합니다:
존댓말/반말 구분
자기소개 패턴
인사 상황별 표현

💡
등록된 교육과정은 학생 진도 추적과 AI 지침서 생성의 기반이 됩니다. 모든 단원을 미리 등록해 두면 진도 관리가 편리합니다.

학습 자료실

ClassIn 클라우드 드라이브에 교재, 영상을 업로드하고 폴더로 관리합니다.

1

폴더 구조 만들기

📁 새 폴더 버튼으로 용도별 폴더를 생성합니다.
예) 교사 교육자료 → 우수수업 영상, 티칭 스킬

2

파일 업로드

📎 파일 업로드 버튼 → 파일 선택 (다중 가능, 최대 500MB)
업로드 진행 상태가 표시되고, 완료 시 토스트 알림이 뜹니다.

3

폴더 탐색

폴더를 클릭하면 내부로 이동합니다. 상단 브레드크럼으로 상위 폴더로 돌아갈 수 있습니다.

수업 녹화

ClassIn 수업 녹화를 가져와서 교사 교육 자료로 활용합니다.

1

녹화 URL 가져오기

수업 녹화 메뉴 → 코스 ID와 수업 ID를 입력 → 녹화 URL 가져오기
ClassIn API에서 자동으로 녹화 재생 URL을 가져옵니다.

2

우수 수업 표시

관리자가 특정 녹화를 ⭐ 우수수업으로 지정하면 모든 강사에게 교육 자료로 공유됩니다.

⚠️
수업 생성 시 녹화(record) 설정이 되어 있어야만 녹화 URL을 가져올 수 있습니다. ClassIn 관리자에게 녹화 기능 활성화를 요청하세요.

진도 관리 / 지침서 / 정산

학생 진도 추적, AI 수업 지침서 생성, 강사 수입 정산

1

학생 진도 업데이트

학생진도/지침 메뉴에서 학생 UID + 교육과정 ID + 상태(예정/진행중/완료)를 입력합니다.
강사가 수업 후 직접 업데이트하거나, 관리자가 일괄 관리할 수 있습니다.

2

AI 수업 지침서

교육과정 ID와 학생 모국어를 입력 → 🤖 AI 지침 생성
Claude AI가 수업 도입, 핵심 포인트, 연습 활동, 모국어별 실수 대처법을 자동 생성합니다.

3

정산/수입

정산/수입 메뉴에서 총 수업 수, 시간, 예상 수입을 확인합니다.
시급은 내 프로필에서 설정한 값으로 자동 계산됩니다.

🔑
AI 지침서를 사용하려면 Render 환경변수에 ANTHROPIC_API_KEY가 설정되어 있어야 합니다.

관리자(원장)에게 받은 로그인 계정으로 접속합니다.

각 카드를 클릭하면 해당 페이지로 바로 이동합니다.

다가오는 수업

오른쪽에 예정된 수업 5개가 시간순으로 표시됩니다.

내 수업

2

수업 클릭 → 입장

링크를 클릭하면 ClassIn 수업에 바로 입장합니다.

학생 진도 / 수업 지침

내 학생들의 진도를 확인하고, AI가 생성한 수업 지침을 참고합니다.

1

내 학생 진도 현황

학생진도/지침 메뉴를 클릭하면 본인에게 배정된 학생들의 진도가 자동으로 표시됩니다.
✅ 완료 👉 진행 중 (오늘 수업) ○ 예정

2

AI 수업 지침서

교육과정 단원 ID와 학생 모국어를 입력 → 🤖 AI 지침 생성
약 10~20초 후 수업 도입, 핵심 포인트, 연습 활동, 모국어별 실수 대처법이 자동 생성됩니다.

3

수업 후 진도 업데이트

수업이 끝나면 해당 단원의 상태를 완료로, 다음 단원을 진행 중으로 변경합니다.
